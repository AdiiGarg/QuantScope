# QuantScope

## Overview
Stock analytics dashboard using Python + C++ (Competitive Programming concepts)

## Features
- Real-time stock data (API)
- Candlestick charts
- Multi-stock comparison
- C++ backend analytics:
  - Prefix Sum
  - Sliding Window
  - Kadane
  - DSU
  - Graph BFS
  - Binary Search
  - Segment Tree
  - etc.

## Tech Stack
- Python (Streamlit)
- C++
- TwelveData API

## How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run dashboard
streamlit run dashboard.py

### 3. Compile C++ engine
g++ cp_engine.cpp -o cp_engine.exe
